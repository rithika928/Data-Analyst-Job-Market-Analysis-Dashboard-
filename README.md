
# 📊 Data Analyst Job Market Analysis Dashboard using Microsoft Excel
## Overview

This project presents an interactive Microsoft Excel dashboard built by analyzing **2,251 Data Analyst job postings**. The objective was to transform raw recruitment data into meaningful business insights through data cleaning, salary extraction, skill analysis, KPI reporting, and interactive visualizations.
The dashboard explores key hiring trends such as salary distribution, top hiring companies, hiring locations, company size, industry demand, and the most in-demand technical skills. It demonstrates practical Excel skills used in real-world business analytics and reporting.

## Key Features

- Cleaned and transformed 2,251 Data Analyst job postings
- Extracted minimum, maximum, and average salary from raw salary estimates
- Identified the most in-demand technical skills from job descriptions
- Built KPI cards for business metrics
- Created Pivot Tables and Pivot Charts
- Designed an interactive Excel dashboard for business insights

## Business Questions Answered

- Which companies hire the most Data Analysts?
- Which locations have the highest demand?
- What is the average and highest salary offered?
- Which industries recruit the most Data Analysts?
- Which technical skills are requested most frequently?
- How does company size vary across job postings?
